# Weet

Weet is a Flutter app for visualizing human relationships.

## Features

- Categorize people as Family, Friends, Business, and custom categories.
- Add people with a relationship score from 0 to 100.
- Use quick checklist chips (20/40/60/80/100) to set closeness fast.
- View a relationship map where higher score = closer to the center.

## Run

```bash
flutter pub get
flutter run
```
